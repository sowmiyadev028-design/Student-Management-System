# 🎓 Student Management System

A Java console-based application that allows users to manage student records.

## ✨ Features

- Add Student
- View Students
- Search Student by ID
- Update Student Details
- Delete Student
- Save Students to File
- Load Students from File
- Exception Handling

## 🛠 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- File Handling
- Exception Handling

## 📚 OOP Concepts Used

- Classes & Objects
- Constructors
- Inheritance
- Polymorphism
- Encapsulation

Compile:

```bash
javac *.java
```

Run:

```bash
java Main
```
